##  Project Overview

The **Career Hub Website** is a static web application designed to help students explore career opportunities, learning resources, and guidance.

This project demonstrates how to deploy a scalable and cost-effective website using Microsoft Azure cloud services.

---

## Azure Services Used

* Azure Blob Storage → Hosting static website
* Static Website Feature → Public access via URL

---

##  Features

* Easy navigation for career resources
